# 음주 일정
술을 마시는 경우 술을 너무 많이 마시는 것은 몸에 좋지 않습니다. 우리의 응용 프로그램은 귀하의 음주 날짜, 소비 된 알코올의 양 및 소비량을 기록하는 것입니다. 음주량을 조절하고 건강한 삶을 누리시기 바랍니다.
